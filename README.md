## Projeto de Doação - Fatec Lins-SP

Projeto desenvolvido para as Aulas de Engenharia de Software I, Programação Web e Algoritmos. 

O trabalho foi feito sobre um sistema de agendamento de doação de sangue.

Optou-se pelo desenvolvimento de uma página em Web, já que no 1º Semestre temos as aulas de Programação Web. 

As tecnologias utilizadas foram apenas HTML e CSS, não possui linguagem de script nem backend, visto que ainda não foram vistas essas linguagens neste primeiro semestre do curso.
